The markup looks something like this:

```html
<figure>
  <img src="/nature.jpg" alt="Nature" />
  <figcaption>Some sample text</figcaption>
</figure>
```

And the styling:

```css
figcaption {
  --lightness: 100%;
  backdrop-filter: blur(10px);
  background-color: hsl(0 0% var(--lightness) / 50%);

  @media (prefers-color-scheme: dark) {
    --lightness: 0%;
  }
}
```
